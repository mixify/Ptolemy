자기 학습 매크로와 사용자 패턴 분석 기반 자동 매크로 생성 프로그램

기존 현존 하는 매크로의 기능들
기능1 : 사용자의 마우스, 키보드 이동을 녹음하는 기능
기능2 : 마우스나 키보드 이동을 스크립트 형식으로 짤수 있는기능
기능3 : 녹음한 것이나 스크립트대로 마우스 키보드 인터럽트를 그대로 실행하는 기능

프로그램에 추가로 하려는 기능
추가 기능1 : 매 실행 시마다 사용자의 피드백 or 화면을 캡쳐등을 통해 하여 긍정적, 부정적
피드백을 받아 마우스 키보드 인터럽트의 delay, 마우스 이동을 자동으로 학습하는
기능
-추가로 어떠한 인터럽트를 넣는다, 화면 인식을해서 어떨때 클릭을 한다...
어떠한 부분에 루프를 돈다 이런것도 사용자가 지정할 수 있도록 해야 할 것
추가 기능2 : 평소에 마우스나 키보드 인터럽트를 계속 로그로 남기며 사용자가 자주
사용하는 패턴을 분석해 이를 매크로나 숏컷으로 만들어 사용할 지 추천해주는 기능
ex)항상 메일을 확인 하기 위해 브라우저를 켜고 구글을 들어가 gmail로 들어가는
사람이 있으면 이를 분석해 자동으로 매크로로 만들어 준다.

현존 API
사용자 마우스, 키보드 녹음 or 스크립트 : pyhook
사용자 마우스, 키보드 재생 : pyautogui
화면캡쳐 : pil
학습 라이브러리 : tensorflow, pytorch...
마우스 키보드 로그 : pynput
